# MedStaff — медицинский агрегатор вакансий

Светлая статическая MVP-версия сайта для проекта MedStaff.

## Что внутри

- Главная страница
- Каталог вакансий с фильтрами
- Медицинские смены
- База специалистов
- Каталог клиник
- Демо-кабинеты работодателя, специалиста и администратора
- Аналитика зарплат
- Тарифы
- FAQ
- Модальные формы
- Адаптивная версия для телефона

## Запуск локально

Откройте `index.html` в браузере.

## Публикация через GitHub Pages

1. Загрузите файлы в репозиторий.
2. Откройте Settings → Pages.
3. Source: Deploy from a branch.
4. Branch: `main`, folder: `/root`.
5. Сохраните настройки.

После этого сайт будет доступен по адресу вида:

`https://username.github.io/repository-name/`

## Следующий этап разработки

Для полноценного запуска нужно добавить backend, базу данных, регистрацию, авторизацию, сохранение форм, загрузку документов, платежи и уведомления.
